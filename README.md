## 🧭 O que é este repositório?

Este repositório funciona como um **laboratório de estudos em Java**, onde desenvolvo **mini projetos independentes** com foco em prática real de conceitos fundamentais e avançados da linguagem.

Cada projeto é pensado como um experimento: pequeno, focado e voltado para aprendizado contínuo, organização de código e boas práticas.

---

## 🎯 Propósito

A ideia central é **aprimorar a base técnica em Java** por meio da construção prática, explorando conceitos como:

- Programação Orientada a Objetos (POO)
- Princípios **SOLID**
- Organização e arquitetura de código
- Boas práticas de desenvolvimento
- Evolução gradual de complexidade

---

## 🧩 Estrutura do repositório

Os projetos são organizados em pastas separadas, onde cada uma representa um **mini projeto isolado**, permitindo:

- Estudar conceitos de forma independente  
- Evoluir projetos sem impactar outros  
- Testar abordagens diferentes para o mesmo problema  

Isso mantém o repositório flexível e fácil de expandir.

---

## 🛠️ Ferramentas & Ambiente

O desenvolvimento dos projetos é feito utilizando:

- **Java**
- **IntelliJ IDEA** como IDE principal

A escolha do IntelliJ se dá pela produtividade, organização e suporte avançado ao ecossistema Java.
